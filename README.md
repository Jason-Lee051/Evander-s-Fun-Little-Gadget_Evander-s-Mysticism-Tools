# Evander's Mysticism Tools 玄学工具箱
🎈这只是一个半成品！目前【奇门遁甲】功能较为完善，【梅花易数】为新增模块。

请谨慎使用，下述 README 未经严格审查后续会进行修正，仅供参考。

基于 PySide6 开发的跨平台玄学排盘工具，目前已实现 **奇门遁甲时家排盘** 与 **梅花易数起卦** 两大术数功能，并集成 LLM 智能解盘分析。

本工具依托天文库精准计算二十四节气，完整实现标准时家奇门排盘逻辑；梅花易数支持数字、时间等多种起卦方式，自动推导本卦、变卦、互卦及体用生克。采用多线程异步计算解决 UI 主线程卡死问题，搭配年份节气缓存大幅提升重复排盘速度。内置可视化九宫盘面与六爻卦象图，支持对接所有兼容 OpenAI 协议的大模型自动断局。项目模块化分层架构，预留扩展接口，后续可新增八字、六爻等更多术数功能。

---
## ✨ 主要特性
- **奇门遁甲完整排盘**
  自动推算节气、局数、三元，生成地盘/天盘/九星/八门/八神全量数据；九宫格可视化区分吉凶色彩（吉绿凶红、三奇金色高亮）。

- **梅花易数灵活起卦**
  支持数字起卦（三数）、时间起卦、汉字起卦（简易字数法），自动计算本卦、变卦、互卦，判定体用五行生克关系，并以六爻图直观展示动爻。

- **AI 智能解盘分析**
  兼容 DeepSeek、OpenAI 等标准 API 大模型，输入预测事项与方位后一键生成通俗断局文本，分析结果支持一键复制。两种术数均使用各自专属提示词模板，分析结果在主窗口右侧流式输出。

- **可视化灵活配置面板**
  图形界面修改 API Key、模型名称、温度等参数；敏感密钥配置文件已加入 `.gitignore`，避免隐私泄露。

- **流畅无阻塞计算**
  使用 QThread 分离 UI 线程与天文计算，搭配节气年份缓存，首次/重复排盘均不会造成窗口卡死。

- **跨平台兼容**
  原生支持 Windows / macOS / Linux 全系统运行。

## 🚀 快速部署
### 1. 克隆仓库 & 初始化虚拟环境
```bash
git clone https://github.com/Jason-Lee051/Evander-s-Fun-Little-Gadget_Evander-s-Mysticism-Tools.git
cd Evander-s-Fun-Little-Gadget_Evander-s-Mysticism-Tools

# 创建虚拟环境
python -m venv venv

# Windows 激活环境
venv\Scripts\activate
# macOS / Linux 激活环境（未经验证是否可行）
source venv/bin/activate

# 安装全部依赖
pip install -r requirements.txt
```

### 2. 配置 LLM（可选，无需 AI 分析可跳过）（程序运行后主窗口-设置中也可以直接输入api信息，更为方便）
复制配置模板生成私有配置文件，填入你的大模型 API 密钥：
```bash
cp config/llm_config.example.json config/llm_config.json
# 使用文本编辑器修改 llm_config.json，填入 API Key、接口地址、模型名
```

### 3. 启动程序
```bash
python main.py
```

## 📖 使用指南
### 奇门遁甲
1. **执行排盘**：顶部菜单「奇门遁甲」→「开始排盘...」，弹窗填写预测事项、所在位置，确认后自动后台计算盘面。
2. **查看九宫盘局**：主窗口渲染完整奇门九宫格，吉门/吉神标记绿色，凶门/凶神标记红色，三奇天干金色突出显示。
3. **AI 智能分析**：排盘完成后点击底部「智能分析」按钮，大模型解读结果在主窗口右侧流式输出，支持全文字复制。

### 梅花易数
1. **开始起卦**：顶部菜单「梅花易数」→「开始起卦...」，弹窗选择起卦方式（数字、时间、汉字）。
   - **数字起卦**：输入三个数字（或报数），程序按“卦以八除，爻以六除”规则计算上卦、下卦、动爻。
   - **时间起卦**：自动使用当前时间计算年、月、日、时（地支）生成卦象。
   - **汉字起卦**：输入一段文字，按字数（或笔画，当前仅实现字数法）起卦。
2. **查看卦象**：主窗口左侧显示六爻图（阳爻 `⚊`、阴爻 `⚋`），高亮动爻，并标注本卦、变卦、互卦名称及体用生克关系。
3. **AI 智能分析**：起卦完成后同样点击「智能分析」按钮，AI 将基于卦象、体用、爻辞等给出百分制评分及详细解读。

### 通用
- **修改 LLM 参数**：顶部菜单「设置」→「LLM API 设置...」，可视化修改密钥、模型、采样温度等参数。

## ⚠️ 注意事项
1. **API 密钥安全**：`config/llm_config.json` 已写入 `.gitignore`，请勿将包含密钥的配置提交至公开仓库。
2. **Python 版本**：运行环境 Python ≥ 3.8。
3. **历法精度说明**：节气计算基于 ephem 天文库，绝大多数年份精准，极少数跨年边界日期存在微小偏差。
4. **梅花易数起卦**：当前仅支持简单的数字、时间和字数起卦，笔画数起卦尚未实现，汉字起卦结果仅供参考。
